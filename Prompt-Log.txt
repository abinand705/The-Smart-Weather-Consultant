3. The Learning


One thing the AI used that I didn't understand at first was:

"const results = await Promise.all(fetchPromises);"

I didn't know what Promise.all() did. After looking it up, I learned that:
Promise.all() used to handle multiple asynchronous operations at the same time. It takes an array of promises (like your API fetch requests), runs them in parallel, and waits until all of them are completed. Once all promises resolve, it returns their results as an array. If any one promise fails, it immediately throws an error.
This makes the page load 3x faster than fetching cities one by one.

So now I understand that Promise.all() is like saying:
"Start all these tasks simultaneously, and tell me when every single one is finished."
